    关于博客

    以前都是离线使用 Obsidian，文件夹 + markdown 组织维护自己的笔记，然后部分写在 CSDN、简书或者公众号上。这是几个搭建博客的理由：

    1. 平台带有社交广告推荐等眼花缭乱的功能，纷繁复杂，不太存粹。
    2. 我的需求：技术开发、生活感悟、读书笔记，文件多碎片化，平台间转储更新同步备份麻烦。
    3. 自由，无拘无束。不会被误删除或者屏蔽，想说啥写多少，没字数审核等限制。
    4. 互联网的开放性，对比离线方式，便于自己随时随地查看或分享。

    所以搭建一个自己的博客，满足表达欲望，就是这个自留地。

    本博客托管在 github，基于 Hexo + GitHub Pages 搭建，主题使用的是 3-hexo 。
    主要功能：

    • 搜索支持文章标题、标签(#标签)、作者(@作者)
    • pad/手机等移动端适配
    • 页面全局快捷键 3-hexo快捷键说明
