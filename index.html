    个人介绍

    您好，我是高俊，现居广州，一个热爱生活和 Coding 的 IT 码仔。

    欢迎来到我的个人博客。这是一个新的尝试，我想将这些年在学校和工作上的计算机知识梳理一下， 搭建自己的知识体系，也督促自己及时思考总结，对知识进行沉淀和分享。

    后续将在博客中按目录分类和标签进行展开，陆续更新，也请看到博客的您多多指教，谢谢。

    我大学主修软件工程专业，2016 年入学，修完四年的课程于 2020 年毕业。毕业后进入 IT 行业从事 Java 研发工作，经过紧张的面试准备，幸运于很快就找到了工作，然后和大家一样经历了三年疫情，民生多艰。

    工作上技术以 Java 周边的技术栈为主，涉及 Linux 服务端web开发，大数据，GIS等，喜欢阅读，还喜欢研究新技术，知识管理，时间管理等方法。

    除了上班勤勤恳恳地敲代码，业余还喜欢犒劳下自己，下厨做美食，外出探店，或者路边喝杯咖啡吃个甜点小憩下。我也很喜欢户外运动，无奈上班以来时间渐少，但还是会忙里偷闲，拿出些时间户外游山玩水，以前也去过很多地方，遇到很多有趣的人和事，后面有兴趣可以写几篇聊聊。

    博客作为自留地，没啥人看，当作自言自语和内心聊天。主要记录：技术开发、生活感悟、读书笔记。
